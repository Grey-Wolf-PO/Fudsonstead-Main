# Fudsonstead Bot
An advanced bot for PS!

Set-up
------
First, install node.
``cd %HOMEPATH%/Desktop/FDS_Bot`` for Windows, ``cd ~/Desktop/FDS_Bot`` for mac. \n
``npm install`` \n
``node main.js [your name]`` \n
You will be given admin priviledges on the bot that way.
