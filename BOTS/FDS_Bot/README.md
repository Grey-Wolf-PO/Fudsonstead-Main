# Fudsonstead Bot
An advanced bot for PS!

Set-up
------
First, install node.
``cd %HOMEPATH%/Desktop/FDS_Bot`` for Windows, ``cd ~/Desktop/FDS_Bot`` for mac.
``npm install``
``node main.js [your name]``
You will be given admin priviledges on the bot that way.
