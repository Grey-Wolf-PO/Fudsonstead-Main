# Fudsonstead Bot
An advanced bot for PS!

Set-up
------
First, install node.
1. ``cd %HOMEPATH%/Desktop/FDS_Bot`` for Windows, ``cd ~/Desktop/FDS_Bot`` for mac.
2. ``npm install``
3. ``node main.js [your name]``
You will be given admin priviledges on the bot that way.
